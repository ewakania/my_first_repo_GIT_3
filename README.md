# Strona Internetowa o Masażu Kobido

## Opis projektu
Projekt ma na celu stworzenie nowoczesnej, estetycznej i responsywnej strony internetowej poświęconej masażowi Kobido – japońskiej technice masażu twarzy znanej ze swoich właściwości liftingujących i relaksacyjnych.  
Strona będzie pełnić funkcję wizytówki online dla gabinetu masażu, prezentując ofertę, informacje o technice Kobido, cennik oraz dane kontaktowe.
